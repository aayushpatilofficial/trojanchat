üöÄ Getting Started Guide

Step 1: Connect

Open this application in any browser. It automatically connects to the server.

Step 2: Type Command

Enter a command using the supported syntax. Commands are validated before sending.

Step 3: Broadcast

The command is instantly broadcast to all connected browsers in real-time.

Step 4: Execute

Each browser receives and executes the command safely within the sandbox.
